Relations represent one to many relationships between elements in AAA and BBB.

Theorem: The number of binary relations on a set A where ∣A∣=n\mid A \mid = n∣A∣=n is: 2n22^{n^2}2n2

Proof:

  • If ∣A∣=n\mid A \mid = n∣A∣=n then the cardinality of the Cartesian product ∣A×A∣=n2\mid A \times A \mid = n^2∣A×A∣=n2.
  • RRR is a binary relation on AAA if R⊆A×AR \subseteq A \times AR⊆A×A (that is, RRR is a subset of A×AA \times AA×A).
  • The number of subsets of a set with kkk elements: 2k2^k2k
  • The number of subsets of A×AA \times AA×A is 2A×A=2n22^{A \times A} = 2^{n^2}2A×A=2n2

Types of Relations

Let’s come up with examples for each.

  • Empty Relation: A relation RRR on a set AAA is called Empty if the set AAA is empty set.
  • Full Relation: A binary relation RRR on a set AAA and BBB is called full if A×BA \times BA×B.
  • Reflexive Relation: A relation RRR on a set AAA is called reflexive if (a,a)∈R(a, a) \in R(a,a)∈R holds for every element a∈Aa \in Aa∈A, e.g. if set A={a,b}A = \{a, b\}A={a,b} then R={(a,a),(b,b)}R = \{(a, a), (b, b)\}R={(a,a),(b,b)} is reflexive relation.
  • Irreflexive relation: A relation RRR on a set AAA is called reflexive if no (a,a)∈R(a, a) \in R(a,a)∈R holds for every element a∈Aa \in Aa∈A, e.g. if set A={a,b}A = \{a, b\}A={a,b} then R={(a,b),(b,a)}R = \{(a, b), (b, a)\}R={(a,b),(b,a)} is irreflexive relation.
  • Symmetric Relation: A relation RRR on a set AAA is called symmetric if (b,a)∈R(b, a) \in R(b,a)∈R holds when (a,b)∈R(a, b) \in R(a,b)∈R, e.g. The relation R={(4,5),(5,4),(6,5),(5,6)}R=\{(4, 5), (5, 4), (6, 5), (5, 6)\}R={(4,5),(5,4),(6,5),(5,6)} on set A={4,5,6}A=\{4, 5, 6\}A={4,5,6} is symmetric.
  • Anti-Symmetric Relation: A relation RRR on a set AAA is called anti-symmetric if (a,b)∈R(a, b) \in R(a,b)∈R and (b,a)∈R(b, a) \in R(b,a)∈R then a=ba = ba=b is called antisymmetric, e.g. The relation R={(a,b)→R∣a≤b}R = \{(a, b) \rightarrow R \mid a \leq b\}R={(a,b)→R∣a≤b} is anti-symmetric since a≤ba \leq ba≤b and b≤ab \leq ab≤a implies a=ba = ba=b.
  • Transitive Relation: A relation RRR on a set AAA is called transitive if (a,b)∈R(a, b) \in R(a,b)∈R and (b,c)∈R(b, c) \in R(b,c)∈R then (a,c)∈R(a, c) \in R(a,c)∈R for all a,b,c∈Aa, b, c \in Aa,b,c∈A, e.g. Relation R={(1,2),(2,3),(1,3)}R=\{(1, 2), (2, 3), (1, 3)\}R={(1,2),(2,3),(1,3)} on set A={1,2,3}A=\{1, 2, 3\}A={1,2,3} is transitive.
  • Equivalence Relation: A relation is an Equivalence Relation if it is reflexive, symmetric, and transitive. E.g. relation R={(1,1),(2,2),(3,3),(1,2),(2,1),(2,3),(3,2),(1,3),(3,1)}R=\{(1, 1), (2, 2), (3, 3), (1, 2), (2, 1), (2, 3), (3, 2), (1, 3), (3, 1)\}R={(1,1),(2,2),(3,3),(1,2),(2,1),(2,3),(3,2),(1,3),(3,1)} on set A={1,2,3}A=\{1, 2, 3\}A={1,2,3} is equivalence relation as it is reflexive, symmetric, and transitive.
  • Asymmetric Relation: Asymmetric relation is opposite of symmetric relation. A relation RRR on a set AAA is called asymmetric if no (b,a)∈R(b, a) \in R(b,a)∈R when (a,b)∈R(a,b) \in R(a,b)∈R.
