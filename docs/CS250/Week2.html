Introduction to Relations

Let A={a1,a2,...,ak}A = \{a_1, a_2, ..., a_k\}A={a1​,a2​,...,ak​} and B={b1,b2,...,bm}B = \{b_1, b_2, ..., b_m\}B={b1​,b2​,...,bm​}

Recall that a Cartesian product A×BA \times BA×B is defined by a set of pairs {(a1,b1),(a1,b2),...,(a1,bm),...,(ak,bm)}\{(a_1, b_1), (a_1, b_2), ..., (a_1, b_m), ..., (a_k, b_m)\}{(a1​,b1​),(a1​,b2​),...,(a1​,bm​),...,(ak​,bm​)}.

A Cartesian product defines a product set, or a set of all ordered arrangements of elements in sets in the Cartesian product.

Binary relations:

If A={1,2}A = \{1, 2\}A={1,2} and B={a,b,c}B = \{a, b, c\}B={a,b,c}, then:

A×B={⟨1,a⟩,⟨1,b⟩,⟨1,c⟩,⟨2,a⟩,⟨2,b⟩,⟨2,c⟩}A \times B = \{\langle 1, a \rangle, \langle 1, b \rangle, \langle 1, c \rangle, \langle 2, a \rangle, \langle 2, b \rangle, \langle 2, c \rangle\}A×B={⟨1,a⟩,⟨1,b⟩,⟨1,c⟩,⟨2,a⟩,⟨2,b⟩,⟨2,c⟩}

A binary relation RRR from AAA to BBB is a subset of A×BA \times BA×B
If ⟨a,b⟩∈R\langle a, b \rangle \in R⟨a,b⟩∈R, then we can write RabRabRab or aRbaRbaRb.
The former is typically preferred, since it allows us an easy convention for notating relationships with >2 relata, e.g. RabcRabcRabc, and to notate the negation ¬Rab\neg Rab¬Rab, using our logical vocabulary.

The inverse relation R−1R^{-1}R−1 is the relation from BBB to AAA which consists of the ordered pairs, which, when reversed, belong to RRR. In symbolic notation: R−1={⟨b,a⟩∣⟨a,b⟩∈R}R^{-1} = \{\langle b, a \rangle | \langle a, b \rangle \in R\}R−1={⟨b,a⟩∣⟨a,b⟩∈R}

Relation Composition or product:
Suppose that we have three sets AAA, BBB, and CCC, a relation RRR defined from AAA to BBB, and a relation SSS defined from BBB to CCC. We can define a composition of RRR and SSS, written (R∣S)(R | S)(R∣S) (but sometimes S∘RS \circ RS∘R), as follows. If aaa is an element of AAA and ccc is an element of CCC, then (R∣S)ac(R | S)ac(R∣S)ac iff there exists some element bbb in BBB such that RabRabRab and SbcSbcSbc. So we have a relation R∣SR|SR∣S from aaa to ccc iff aaa is RRR related to bbb and bbb is SSS related to ccc.
