CS250 at CCUT Week 2: Relations

You should have had an introduction to set theory by now. Let’s do a quick recap and take a look at the next topic: relations.

Sets

  • A set is an unordered collection of objects, e.g., students in this class; air molecules in this room.
  • The objects in a set are called the elements, or members of the set. A set is said to contain its elements.
  • The notation x∈Sx \in Sx∈S denotes that xxx is an element of the set SSS.
  • If xxx is not a member of SSS, write x∉Sx \notin Sx∉S.

Describing sets

  • S={a,b,c,d}S = \{a, b, c, d\}S={a,b,c,d}.
  • Order not important S={a,b,c,d}={b,c,a,d}S = \{a, b, c, d\} = \{b, c, a, d\}S={a,b,c,d}={b,c,a,d}.
  • Each distinct object is either a member or not; listing more than once does not change the set. S={a,b,c,d}={a,b,c,b,c,d}S = \{a, b, c, d\} = \{a, b, c, b, c, d\}S={a,b,c,d}={a,b,c,b,c,d}.
  • Dots “. . . ” may be used to describe a set without listing all of the members when the pattern is clear. S={a,b,c,d,...,z}S = \{a, b, c, d, . . . , z\}S={a,b,c,d,...,z} or S={5,6,7,...,20}S = \{5, 6, 7, . . . , 20\}S={5,6,7,...,20}.
  • Try not to overuse this. Patters are not always as clear as the writer thinks

Set Builder Notation

Another way of describing a set is to specify the property (or properties) that all members of the set must satisfy. We use a pipe, '∣\mid∣’, to read "such that."

  • S={x∣S = \{x \midS={x∣ x is a positive integer less than 100 }\}}
  • S={x∣x∈Z+S = \{x \mid x \in \mathbb{Z}+S={x∣x∈Z+ and x<100}x < 100\}x<100}
  • S={x∈Z+∣x<100}S = \{x \in \mathbb{Z}+ \mid x < 100\}S={x∈Z+∣x<100}
  • A predicate can be used, e.g., S={x∣P(x)}S = \{x \mid P(x)\}S={x∣P(x)} where P(x)P(x)P(x) is true iff xxx is a prime number.
  • Positive rational numbers Q+={x∈R∣∃p,q∈Z+x=p/q}\mathbb{Q}+ = \{x \in \mathbb{R} \mid \exists p, q \in \mathbb{Z}+ x = p/q\}Q+={x∈R∣∃p,q∈Z+x=p/q}

Universal Set and Empty Set

  • The universal set UUU is the set containing everything currently under consideration.
    • Content depends on the context.
    • Sometimes explicitly stated, sometimes implicit.
  • The empty set is the set with no elements.
    Symbolized by ∅\varnothing∅ or {}\{\}{}.

Things to remember

Subsets and Set Equality

Definition:
Set AAA is a subset of set BBB iff every element of AAA is also an element of
BBB.

  • Formally: A⊆B↔∀x(x∈A→x∈B)A \subseteq B \leftrightarrow \forall x(x \in A \rightarrow x \in B)A⊆B↔∀x(x∈A→x∈B)
  • In particular, ∅⊆S\varnothing \subseteq S∅⊆S and S⊆SS \subseteq SS⊆S for every set SSS.

Definition:
Two sets A and B are equal iff they have the same elements.

  • Formally: A=B↔A⊆B∧B⊆AA = B \leftrightarrow A \subseteq B \land B \subseteq AA=B↔A⊆B∧B⊆A.
  • Example: {1, 5, 5, 5, 3, 3, 1} = {1, 3, 5} = {3, 5, 1}.

Proper Subsets

Definition:
A is a proper subset of B iff A ⊆ B and A 6= B.

  • This is denoted by A⊂BA \subset BA⊂B.
  • A⊂BA \subset BA⊂B can be expressed by
    ∀x(x∈A→x∈B)∧∃x(x∈B∧x∉A)\forall x(x \in A \rightarrow x \in B) \land \exists x(x \in B \land x \notin A)∀x(x∈A→x∈B)∧∃x(x∈B∧x∉A)

Set Cardinality

Definition:
If there are exactly nnn distinct elements in a set SSS, where nnn is a nonnegative integer, we say that SSS is finite. Otherwise it is infinite.

Definition
The cardinality of a finite set SSS, denoted by ∣S∣|S|∣S∣, is the number of
(distinct) elements of SSS.

Examples:

  • ∣∅∣=0|\varnothing| = 0∣∅∣=0
  • Let S be the set of letters of the English alphabet. Then ∣S∣=26|S| = 26∣S∣=26.
  • ∣{1,2,3}∣=3|\{1, 2, 3\}| = 3∣{1,2,3}∣=3
  • ∣{∅}∣=1|\{\varnothing\}| = 1∣{∅}∣=1
  • The set of integers Z is infinite.

Power Sets

Definition
The set of all subsets of a set SSS is called the power set of SSS.

  • It is denoted by ℘(S)\wp (S)℘(S).
  • Formally: ℘(S)={S′∣S′⊆S}\wp(S) = \{S' \mid S' \subseteq S\}℘(S)={S′∣S′⊆S}
  • In particular, S∈℘(S)S \in \wp(S)S∈℘(S) and ∅∈℘(S)\varnothing \in \wp(S)∅∈℘(S).
  • Example: ℘({a,b})={∅,{a},{b},{a,b}}\wp(\{a, b\}) = \{\varnothing, \{a\}, \{b\}, \{a, b\}\}℘({a,b})={∅,{a},{b},{a,b}}
  • If ∣S∣=n|S| = n∣S∣=n then ∣℘(S)∣=2n|\wp(S)| = 2^n∣℘(S)∣=2n

Introduction to Relations

Let A={a1,a2,...,ak}A = \{a_1, a_2, ..., a_k\}A={a1​,a2​,...,ak​} and B={b1,b2,...,bm}B = \{b_1, b_2, ..., b_m\}B={b1​,b2​,...,bm​}

Recall that a Cartesian product A×BA \times BA×B is defined by a set of pairs {(a1,b1),(a1,b2),...,(a1,bm),...,(ak,bm)}\{(a_1, b_1), (a_1, b_2), ..., (a_1, b_m), ..., (a_k, b_m)\}{(a1​,b1​),(a1​,b2​),...,(a1​,bm​),...,(ak​,bm​)}.

A Cartesian product defines a product set, or a set of all ordered arrangements of elements in sets in the Cartesian product.

Binary relations

Definition: Let AAA and BBB be two sets. A binary relation from AAA to BBB is a subset of a Cartesian product A×BA \times BA×B.

  • Let R⊆A×BR \subseteq A \times BR⊆A×B mean RRR is a set of ordered pairs of the form (a,b)(a, b)(a,b) where a∈Aa \in Aa∈A and b∈Bb \in Bb∈B.
  • We use the notation aRbaRbaRb or sometimes RabRabRab to denote (a,b)∈R(a, b) \in R(a,b)∈R.
  • If aRbaRbaRb, we say that aaa is related to bbb by RRR.

Example: Let A={a,b,c}A=\{a, b, c\}A={a,b,c} and B={1,2,3}B = \{1, 2, 3\}B={1,2,3}.

  • Is R={(a,1),(b,2),(c,2)}R = \{(a, 1), (b, 2), (c, 2)\}R={(a,1),(b,2),(c,2)} a relation from AAA to BBB?
  • Is Q={(1,a),(2,b)}Q = \{(1, a), (2, b)\}Q={(1,a),(2,b)} a relation from AAA to BBB?
  • Is P={(a,a),(b,c),(b,a)}P = \{(a, a), (b, c), (b, a)\}P={(a,a),(b,c),(b,a)} a relation from AAA to AAA?
