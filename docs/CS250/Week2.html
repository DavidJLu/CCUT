Relations have many interesting properties. Some that we're interested in include: reflexivity, symmetry, antisymmetry, transitivity, equivalence, and partial and total order.

Relationship Properties

Let's come up with examples for each.

  • Empty Relation: A relation RRR on a set AAA is called Empty if the set AAA is empty set.
  • Full Relation: A binary relation RRR on a set AAA and BBB is called full if A×BA \times BA×B.
  • Reflexive Relation: A relation RRR on a set AAA is called reflexive if (a,a)∈R(a, a) \in R(a,a)∈R holds for every element a∈Aa \in Aa∈A, e.g. if set A={a,b}A = \{a, b\}A={a,b} then R={(a,a),(b,b)}R = \{(a, a), (b, b)\}R={(a,a),(b,b)} is reflexive relation.
  • Irreflexive relation: A relation RRR on a set AAA is called reflexive if no (a,a)∈R(a, a) \in R(a,a)∈R holds for every element a∈Aa \in Aa∈A, e.g. if set A={a,b}A = \{a, b\}A={a,b} then R={(a,b),(b,a)}R = \{(a, b), (b, a)\}R={(a,b),(b,a)} is irreflexive relation.
  • Symmetric Relation: A relation RRR on a set AAA is called symmetric if (b,a)∈R(b, a) \in R(b,a)∈R holds when (a,b)∈R(a, b) \in R(a,b)∈R, e.g. The relation R={(4,5),(5,4),(6,5),(5,6)}R=\{(4, 5), (5, 4), (6, 5), (5, 6)\}R={(4,5),(5,4),(6,5),(5,6)} on set A={4,5,6}A=\{4, 5, 6\}A={4,5,6} is symmetric.
  • Anti-Symmetric Relation: A relation RRR on a set AAA is called anti-symmetric if (a,b)∈R(a, b) \in R(a,b)∈R and (b,a)∈R(b, a) \in R(b,a)∈R then a=ba = ba=b is called antisymmetric, e.g. The relation R={(a,b)→R∣a≤b}R = \{(a, b) \rightarrow R \mid a \leq b\}R={(a,b)→R∣a≤b} is anti-symmetric since a≤ba \leq ba≤b and b≤ab \leq ab≤a implies a=ba = ba=b.
  • Transitive Relation: A relation RRR on a set AAA is called transitive if (a,b)∈R(a, b) \in R(a,b)∈R and (b,c)∈R(b, c) \in R(b,c)∈R then (a,c)∈R(a, c) \in R(a,c)∈R for all a,b,c∈Aa, b, c \in Aa,b,c∈A, e.g. Relation R={(1,2),(2,3),(1,3)}R=\{(1, 2), (2, 3), (1, 3)\}R={(1,2),(2,3),(1,3)} on set A={1,2,3}A=\{1, 2, 3\}A={1,2,3} is transitive.
  • Equivalence Relation: A relation is an Equivalence Relation if it is reflexive, symmetric, and transitive. E.g. relation R={(1,1),(2,2),(3,3),(1,2),(2,1),(2,3),(3,2),(1,3),(3,1)}R=\{(1, 1), (2, 2), (3, 3), (1, 2), (2, 1), (2, 3), (3, 2), (1, 3), (3, 1)\}R={(1,1),(2,2),(3,3),(1,2),(2,1),(2,3),(3,2),(1,3),(3,1)} on set A={1,2,3}A=\{1, 2, 3\}A={1,2,3} is equivalence relation as it is reflexive, symmetric, and transitive.
  • Asymmetric Relation: Asymmetric relation is opposite of symmetric relation. A relation RRR on a set AAA is called asymmetric if no (b,a)∈R(b, a) \in R(b,a)∈R when (a,b)∈R(a,b) \in R(a,b)∈R.

Functions Examples and Exercises

Some definitions first.

Functions are ordinarily denoted by symbols. Suppose that for each element of a set AAA, the domain, we assign a unique element of a set BBB, the codomain. The collection of these assignments is called a function from AAA to BBB. Let fff denote that function. We write:

f:A→Bf: A \to Bf:A→B

We can also say that fff maps AAA to BBB.

Functions are often expressed by means of a mathematical formula. For example, consider the function which maps each real number to its square. We can denote this function by writing

  • f(x)=x2f(x) = x^2f(x)=x2
  • x↦x2x \mapsto x^2x↦x2
  • y=x2y = x^2y=x2

In the first notation, xxx is called a variable or argument and the letter fff denotes the function.
In the second, the barred arrow ↦\mapsto↦ is read "goes to" or "maps to". (The LaTeX command is \mapsto).
In the last, xxx is called the independent variable and yyy is called the dependent variable since the value of yyy depends on the value of xxx.

Functions as Relations

Every function f:A→Bf: A \to Bf:A→B defines a relation from AAA to BBB called the graph of fff

Graph of f={⟨a,b⟩∣a∈A,f(a)=b∈B}f = \{\langle a, b \rangle | a \in A,f(a) = b \in B\}f={⟨a,b⟩∣a∈A,f(a)=b∈B}

Two functions f:A→Bf: A \to Bf:A→B and g:A→Bg: A \to Bg:A→B are equal if for every a∈Aa \in Aa∈A, f(a)=g(a)f(a) = g(a)f(a)=g(a), i.e. they have the same graph.

Classes of Functions

A function is injective (one-to-one) if each element of the codomain is mapped to by at most one element of the domain.
That is, for all x,x′∈Ax, x' \in Ax,x′∈A, f(x)=f(x′)→x=x′f(x) = f(x') \to x = x'f(x)=f(x′)→x=x′

A function is surjective (onto) if each element of the codomain is mapped to by at least one element of the domain.
That is, for all y∈By \in By∈B, there exists an x∈Ax \in Ax∈A such that f(x)=yf(x) = yf(x)=y

A function is bijective (one-to-one and onto or one-to-one correspondence) if each element of the codomain is mapped to by exactly one element of the domain.
