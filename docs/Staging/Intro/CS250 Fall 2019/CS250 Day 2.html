Review of Basic Set Theory

Definition 1: A set is a well defined collection of objects, considered as an object itself.

The objects making up the set are called elements or members of the set.

There is no order to the elements and no multiplicity.

If a is an element of set X, then we write a∈Xa \in Xa∈X. And if not, we write a∉Xa \notin Xa∈/​X.

Sets are conventionally denoted with capital letters.

There is a special set with no members, and it's called the empty set, denoted by ∅\emptyset∅.

There are two ways of describing a set.

  • The first is by extension, listing each member of the set. For example: T={1,2,3,4}T = \{1, 2, 3, 4\}T={1,2,3,4}

  • The other is by intensional definition, using a rule or semantic description. For example: SSS is the set containing the siblings of Richard. (Written S={xS = \{xS={x : xxx is a sibling of Richard}\}})

We often have the choice of specifying a set either intensionally or extensionally. For example: R={xR = \{xR={x : xxx is one of the first four positive integers}\}}. So T=RT = RT=R. We use a pipe, '∣\mid∣', or sometimes a colon to read "such that."

Definition 2: If XXX and YYY are sets and have the same elements, then they are identical. (Extensionality) The identity of sets are given by their extensions (the elements they contain). This definition gives us a common proof strategy for proving set identities.

Definition 3: If every element of XXX is an element of YYY, then XXX is a subset of YYY. We write X⊆YX \subseteq YX⊆Y.

It follows from this definition that every set is a subset of itself.

Also, it's possible for a set to be an element of another set.

If X⊆YX \subseteq YX⊆Y and Y⊈XY \not\subseteq XY​⊆X then X⊂YX \subset YX⊂Y. In other words, if every element of XXX is an element of YYY, but not every element of YYY is an element of XXX, then we say that XXX is a proper subset of YYY.

Set Cardinality

Definition:
If there are exactly nnn distinct elements in a set SSS, where nnn is a nonnegative integer, we say that SSS is finite. Otherwise it is infinite.

Definition
The cardinality of a finite set SSS, denoted by ∣S∣|S|∣S∣, is the number of
(distinct) elements of SSS.

Examples:

  • ∣∅∣=0|\varnothing| = 0∣∅∣=0

Why is it important to learn about sets?

We build many other ideas from sets and use them to solve problems mathematically, such as relations, functions, graphs, spaces, etc...

Set theory is foundational to mathematics and computer science.
